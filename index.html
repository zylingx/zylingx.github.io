样式选择器

发表于 2018-10-10 | 分类于 CSS |

星号选择器

将样式应用于所有的标签，慎用

*{margin:0;padding:0}

标签选择器

建议尽量应用在层级选择器中,不建议直接给标签使用

div{color:red}   


<div>
    ....
</div>

eg:下例演示标签选择器的使用，但不建议这样使用

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS页面引入_嵌入式</title>

    <style>
        div{
            width: 800px;
            height: 500px;
            color: gold;

            font-size: 24px;
            font-family: '微软雅黑';
            text-align: center;
        }
    </style>
</head>

<body>
    <div>
        CSS是 Cascading Style Sheets 的首字母缩写，意思是层叠样式表。<br />

        为了让网页元素的样式更加丰富，也为了让网页的内容和样式能拆分开，CSS由此思想而诞生。<br />

        有了CSS，html中大部分表现样式的标签就废弃不用了，html只负责文档的结构和内容，表现形式完全交给CSS，html文档变得更加简洁。<br />
    </div>
</body>
</html>

id选择器

id选择器以#号开头，通过id名来选择元素，元素的id名称不能重复，所以一个样式设置项只能对应于页面上一个元素，不能复用，id名一般给程序使用，所以不推荐使用id作为选择器。

#box{color:red} 

<div id="box">
    ....
</div>   <!-- 对应以上一条样式，其它元素不允许应用此样式 -->

eg:下例演示id选择器的使用，但不建议这样使用

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>选择器</title>

    <style type="text/css">
        #box{
            color:red;
        }
    </style>
</head>
<body>
    <div id="box">
        CSS是 Cascading Style Sheets 的首字母缩写，意思是层叠样式表。<br />

        为了让网页元素的样式更加丰富，也为了让网页的内容和样式能拆分开，CSS由此思想而诞生。<br />

        有了CSS，html中大部分表现样式的标签就废弃不用了，html只负责文档的结构和内容，表现形式完全交给CSS，html文档变得更加简洁。<br />
    </div>
</body>
</html>

类选择器

通过类名来选择元素，一个类可应用于多个元素，一个元素上也可以使用多个类，可复用，是css中应用最多的一种选择器。

.red{color:red}
.big{font-size:20px}
.mt10{margin-top:10px} 

<div class="red">
    ....
</div>

<h1 class="red big mt10">
    ....
</h1>

<p class="red mt10">
    ....
</p>

eg：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>类选择器</title>
    <style type="text/css">
        .red{color:red}
        .big{font-size:20px}
        .mt10{margin-top:10px} 
    </style>
</head>
<body>
    <div class="red">
        CSS是 Cascading Style Sheets 的首字母缩写，意思是层叠样式表。<br />
    </div>

    <div class="mt10 big">
        为了让网页元素的样式更加丰富，也为了让网页的内容和样式能拆分开，CSS由此思想而诞生。<br />
    </div>    

    <div class="red big mt10">
        有了CSS，html中大部分表现样式的标签就废弃不用了，html只负责文档的结构和内容，表现形式完全交给CSS，html文档变得更加简洁。<br />
    </div>
</body>
</html>

层级选择器

主要应用在选择父元素下的子元素，或者子元素下面的子元素，可与标签元素结合使用，减少命名，同时也可以通过层级，防止命名冲突。

.box span{color:red}
.box .red{color:pink}
.red{color:red}

<div class="box">
    <span>....</span>
    <a href="#" class="red">....</a>
</div>

<h3 class="red">....</h3>

eg: 层级选择器的层数最好不要超过4层，层数越少越好，层数越多，解析的越慢

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>层级选择器</title>

    <style type="text/css">
        .list li{
            height: 30px;
            list-style: none;
        }
        .list .color{
            color: red;
        }

        .list a{
            text-decoration: none;
            color: gold;
        }
    </style>
</head>
<body>
    <ul class="list">
        <li><a href="#"> 严重脱发，难道就只能剃头了吗？</a></li>
        <li ><a href="#" class="color"> 豆芽怎么种？两种高大上的种植方法，让你在家也可以种出美味！</a></li>
        <li><a href="#"> 一部荣耀畅玩8C等于几部iPhone XS Max?</a></li>
    </ul>
</body>
</html>

组选择器

多个选择器，如果有同样的样式设置，可以使用组选择器。

.box1,.box2,.box3{width:100px;height:100px}
.box1{background:red}
.box2{background:pink}
.box2{background:gold}

<div class="box1">....</div>
<div class="box2">....</div>
<div class="box3">....</div>

eg:选择器之间用逗号隔开，简化样式的写法

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>组选择器</title>

    <style type="text/css">
        .box1,.box2,.box3{
            font-size: 24px;
            height: 30px;
        }
        .box1{
            color: red;
        }
        .box2{
            color: gold;

        }
        .box3{
            font-weight: bold;
        }
    </style>
</head>
<body>
    <div class="box1">hello world</div>
    <div class="box2">hello world</div>
    <div class="box3">hello world</div>
</body>
</html>

伪类选择器

常用的伪类选择器有hover，表示鼠标悬浮在元素上时的状态

.box1:hover{color:red}

<div class="box1">....</div>

eg:伪类选择器，常与 a标签搭配使用，设置鼠标悬停在元素上的状态，如下面例子中，设置 a标签的颜色

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>伪类选择器</title>

    <style type="text/css">
        .list li{
            height: 30px;
            list-style: none;
        }
        .list a:hover{
            color: gold;
            text-decoration: none;
        }
    </style>
</head>
<body>
    <ul class="list">
        <li><a href="#"> 严重脱发，难道就只能剃头了吗？</a></li>
        <li ><a href="#"> 豆芽怎么种？两种高大上的种植方法，让你在家也可以种出美味！</a></li>
        <li><a href="#"> 一部荣耀畅玩8C等于几部iPhone XS Max?</a></li>
    </ul>
</body>
</html>

伪元素选择器

伪元素选择器有before和after,它们可以通过样式在元素中插入内容。

.box1:before{content:'行首文字';}
.box2:after{content:'行尾文字';}

<div class="box1">....</div>
<div class="box2">....</div>

eg:伪元素选择器是通过样式在文字的前面或后面插入文字，常与content属性配合使用，插入的文字无法被选中

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>伪元素选择器</title>

    <style type="text/css">
        .list .box1:before{
            content: "在前面插入文字 ";
            color: red;
        }

        .list .box2:after{
            content: " 在后面插入文字";
            color: blue;
        }
    </style>
</head>
<body>
    <ul class="list">
        <li class="box1">新闻一</li>
        <li class="box2">新闻二</li>
    </ul>
</body>
</html>

注：伪元素选择器一般用来插入个小图片什么的

常用的选择器大概就这么多，不常用的就不介绍了，更多花里胡哨的选择器将在CSS3里学习
