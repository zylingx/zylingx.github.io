定位

发表于 2018-10-18 | 分类于 CSS |
将学习相对定位、绝对定位、固定定位和定位元素的层级

我们可以使用css的position属性来设置元素的定位类型，postion的设置项如下：

relative 生成相对定位元素，元素所占据的文档流的位置不变，元素本身相对文档流的位置进行偏移
absolute 生成绝对定位元素，元素脱离文档流，不占据文档流的位置，可以理解为漂浮在文档流的上方，
    相对于上一个设置了相对或者绝对或者固定定位的父级元素来进行定位，如果找不到，则相对于body元素进行定位。

fixed 生成固定定位元素，元素脱离文档流，不占据文档流的位置，可以理解为漂浮在文档流的上方，相对于浏览器窗口进行定位。
static 默认值，没有定位，元素出现在正常的文档流中，相当于取消定位属性或者不设置定位属性
inherit 从父元素继承 position 属性的值

案例

相对定位

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>相对定位</title>

    <style type="text/css">
        .con{
            width: 500px;
            height: 300px;
            margin: 0px auto 0px;
            border: 1px solid #000;
        }
        .box1{
            position: relative;
            background-color: gold;
            left: 15px;
            top: 15px;
        }
    </style>
</head>
<body>
    <div class="con">
        <div class="box1">relative 生成相对定位元素，元素所占据的文档流的位置不变，元素本身相对文档流的位置进行偏移</div>
    </div>
</body>
</html>

定位元素特性

绝对定位和固定定位的块元素和行内元素会自动转化为行内块元素

定位元素层级

定位元素是浮动的正常的文档流之上的，可以用z-index属性来设置元素的层级

案例

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>元素层级</title>

    <style type="text/css">
        .con{
            width: 500px;
            height: 300px;
            margin: 0px auto 0px;
            border: 1px solid #000;
            position: relative;
        }
        .box1,.box2,.box3{width: 300px;height: 100px;}
        .box1{
            background-color: gold;
            z-index: 10;
            position: absolute;
            left:30px;
            top: 30px;
        }
        .box2{
            background-color: red;
            position: absolute;
            left:60px;
            top: 60px;
        }
        .box3{
            background-color: pink;
            position: absolute;
            left:90px;
            top: 90px;
        }

    </style>
</head>
<body>
    <div class="con">
        <div class="box1">定位元素是浮动的正常的文档流之上的，可以用z-index属性来设置元素的层级</div>
        <div class="box2">第一个div默认层级是0，第二个div默认层级是1，第三个div默认层级是2</div>
        <div class="box3">现在将第一个div的层级设置为10</div>
    </div>
</body>
</html>
