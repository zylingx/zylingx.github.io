基础知识

发表于 2018-08-26 |
这些基础知识基本有在前面的Python知识里有说过，但是还是选择在这里把基础知识在过一遍 
赋值：将定义好的数据，传递给变量
变量：本义指其值可以改变的量，这里说的是数据赋值的对象，我们通过变量名来操作数据

举例

x = 10
y = x-7
z = 'string'
t = True

变量的命名规则

变量名可以由字母、数字、下划线组成，但不能以数字开头
区分字母的大小写
不能与系统关键字重名 如：

有效标识符名称的例子有i、__my_name、name_23和a1b2_c3。  
无效标识符名称的例子有2things、this is spaced out和my-name。

许多系统变量以下划线开头，所以最好不用下划线开头命名变量

数据类型

逻辑型

又叫布尔型，用于只有两种取值（真 or 假， 0 or 1， 开 or 关等）的场合

值            注释
True        真
False        假

运算规则

运算符    注释        运算规则
  &      按位与     两个逻辑型数据中，有一个逻辑型数据为假，结果为假
  |      按位或     两个逻辑型数据中，有一个逻辑型数据为真，结果为真
  not    取反       取相反的值，真为假，假为真

关于数字间的逻辑比较请看python基础里面的运算符一章

数值型

运算规则

运算符    注释        运算规则
  +      加        
  -      减        
  *      乘        
  /      除       
  %      求余        
  //     整除     只保留整数部分
  **     乘方     指数幂运算

注意

a = 4.2
b = 2.1
print(a+b)

打印

6.300000000000001

小数的运算是浮点数的运算，精度会有一点点的偏差，每种编程语言都有这个问题，在Python中的解决方法

from decimal import Decimal
a = Decimal('4.2')
b = Decimal('2.1')
print(a+b)

打印

6.3

字符型

字符型数据代表了所有可定义的字符

定义方式：使用 ‘’ 或 “” 或 “”””””将其包含起来

'hello world'

"""
    床前明月光
    疑是地上霜
"""

注意：

'' 和 ""之间没有区别,''' ''' 和 """ """之间没有区别
''表示单行字符串，''' '''可以表示多行字符串

转义字符

使用反斜杠（\）+ 字符 可以表示特殊字符 如 \n、 \t等，具体有哪些特殊字符可以在前面的Python基础里的数据类型中或网上可以查到

print("hello \t world!")

原始字符

如果不想使用反斜杠转义，可以在字符串前面加上一个 (r) 表示原始字符串

path = r'C:\windows\python     #注意末尾不能有反斜杠，否则会报错

续行符

反斜杠还可以当做续行符，表示下一行是上一行的延续

s = 'abcdefg \
   higklmn'

多行符

分号表示前面的语句是一个完整的语句，多个命令可以写在同一行

a = 4;b = 5; c = a+b;

当然，我不建议这样写

加号与乘号

加号可以将两个字符串连接在一起，乘号可将字符串重复

s = 'abc'+'def'
st = 'abc' * 3
print(s,st)

打印

'abcdef'    'abcabcabc'

###字符串的索引
Python中的索引方式有两种：一种是从左到右，从0开始依次增加；一种是从右到左，从-1开始依次减少

word = 'Python'
print(word[0], word[5])
print(word[-1], word[-6])

###字符串的切片
对字符串进行切片可以得到一段子串，截取的范围是左闭右开区间

s = 'helloworld'
print(s[1:5])
print(s[:])
print(s[5:])
print(s[:-1])

print(s[1:5:2])

打印

ello
helloworld
world
helloworl
el

详细内容看Python基础里的切片知识

注：不能改变字符串里的内容

s = 'helloworld'
s[2] = 'a'    #error

检测开头和结尾

S.endswith(suffix[, start[, end]]) -> bool
S.startswith(prefix[, start[, end]]) -> bool

案例

Web = 'https://www.zylin.xyz'
Web.endswith('.xyz')        #检测结尾，如果符合返回True
Web.startswith('http:')        #检测开头，如果符合返回True

choices = ('http:','https:') 注意，这仅能接受一个字符元组，不支持列表
Web.startswith(choices)        #只要开头符合两者之一就返回True

查找字符串

S.find(sub[, start[, end]]) -> int

案例

Web = '花木成畦手自栽 https://www.zylin.xyz https:开朗大方'
result = Web.find('https:')        #从左至右查找字符，查找到返回字符的当前位置，查找不到返回-1
print(result)        #打印 8     ，汉字算一个字符
result = Web.rfind('https:')    #从右至左查找字符，和上面一样
print(result)        #打印30

忽略大小写查找字符串

findall(pattern, string, flags=0) -> list

案例

import re    #需要导入re包

Web = 'UPPER PYTHON, lower python, Mixed Python'
re.findall('python',Web, flags = re.IGNORECASE)        #IGNORECASE是一个标志

打印

['PYTHON', 'python', 'Python']

想要详细了解需要学习Python的 正则表达式

替换字符串

S.replace(old, new[, count]) -> str        #不写count参数则默认替换所有字符

案例

text = 'hello world,hello python, hello anaconda'
text.replace('hello','hi')

打印

'hi world,hi python, hi anaconda'

忽略大小写替换字符串

sub(pattern, repl, string, count=0, flags=0) -> str

案例

import re    #需要导入re包
Web = 'UPPER PYTHON, lower python, Mixed Python'
re.sub('python', 'text',Web, flags = re.IGNORECASE)

打印

'UPPER text, lower text, Mixed text'

合并拼接字符串

S.join(iterable) -> str

案例

parts = ['My', 'name', 'is', 'Zylin', 'and','I', 'am', 'a', 'student']

''.join(parts)
' '.join(parts)
','.join(parts)

打印

'MynameisZylinandIamastudent'
'My name is Zylin and I am a student'
'My,name,is,Zylin,and,I,am,a,student'
