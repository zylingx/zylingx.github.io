费曼学习方法

发表于 2018-07-16 | 分类于 学习方法 |
最近学习了很多东西，总是要不停的去回顾，不然很可能过几天就忘记干净了。很苦恼，一定是我方法不对。有什么好的学习方法呢？古人说过:公欲善其事，必先利其器。这句话还是非常有道理的，于是开始上网查找资料，终于有了些许收获。

第一个要介绍的是一个顶尖的学习方法：就是一个标题上的费曼学习法(图：费曼本人)

先看看关于学习的分类有两种：一种是主动学习，一种是被动学习，这涉及到哈佛大学的学习吸收率金字塔

可以看到，主动学习与被动学习的差距。而费曼学习法的原理，就是如何将别人的知识转化为自己的知识的过程,下面附上费曼学习法的流程图：

在预习、复习的过程中，遇到一个重要的、从没见过的新术语新概念跳出来了。拿出一张白纸，在页顶记下它，这是你想要学习的概念。

图中还有一个没有提到的要点——反思。反思你在思考的时候在哪里卡了壳，着重这个地方，再次理解。这个过程也是至关重要!

费曼学习法可以考量自己对一个概念或知识是否理解，一个最直观的方法就是讲给别人听，如果能够用简洁清晰的语言让别人完全听懂你在说什么，就证明你完全了解所学的概念;反之则还需要再去认真学习。
